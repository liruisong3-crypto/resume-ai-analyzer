# AI 赋能的智能简历分析系统

![系统截图](screenshot.png)

## 项目概述
基于阿里云 Serverless + Python 的智能简历分析系统，能够自动解析PDF简历，提取关键信息，并与岗位需求进行智能匹配评分。

##  在线演示
- **前端界面**: [https://yourusername.github.io/resume-ai-analyzer/](https://yourusername.github.io/resume-ai-analyzer/)
- **API文档**: [API文档](./API_DOCUMENTATION.md)

##  核心功能
1. **简历上传与解析**: 支持PDF格式，多页解析，文本清洗
2. **AI信息提取**: 姓名、联系方式、技能、工作经验等
3. **智能匹配评分**: 岗位需求匹配，综合评分
4. **Redis缓存**: 提升响应速度，避免重复处理
5. **响应式前端**: 现代化UI，支持移动端
